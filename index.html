Hosting
All files must live in the same GitHub repository root. Enable GitHub Pages under Settings → Pages → Source: main branch, root folder. The hub references all other files by relative path — no absolute URLs needed.
Updating a Resource
Upload a new version of any HTML file to the repository with the exact same filename. GitHub Pages will serve the updated file within ~60 seconds. The hub requires no changes.
